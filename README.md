### Формулы
В LaTeX-е множество различных математических и не только формул, которые помогут изложить материал правильно и грамотно.
### Списки
Существует осовные виды списков нумерованный и маркированный списки и еще их комбинация.
### Таблицы
Можно создать таблицы различного уровня с разными размерами ячеек + можно добавлять различные цвета в таблицы, прямо как в CSS.
### Картинки
В свои работы можно добавлять картинки и менять их размер под стиль текста.
### Литература
Можно добавлять цитирование из различных источников, а все используемые источники будут отображаться в разделе с Литературой, причем LaTeX следит за порядком добавления цитирования, так что не придется вручную добавлять нумерацию источников.

# Туториал по LaTeX
## Установка
Чтобы писать работы необходим IDE, удобнее всего использовать онлайн редакторы по типу overleaf. Однако существует и оффлайн редакторы, которые можно скачать на следующие операционные системы:
- Windiws
- MacOS
- Linux

## Заголовки
Чтобы правильно структурировать свои работы в LaTeX-е существуют команды для создание заголовков и подзаголовков с автоматической нумерацией.
```LaTeX
    \section{}
    \subsection{}
    \subsubsection{}

    \paragraph{}
    \subparagraph{}
```
## Пакеты
Чтобы импортировать пакет в LaTeX, нужно просто добавить строку \usepackage{ИМЯ ПАКЕТА} в документе. Пакеты обычно используются для упрощения и разнообразия используемых функций, например существует пакет equation для математического режима.

## Формулы
Чтобы использовать формулы в тесте, достаточно написать код формлуы со слэшем и обернуть все в знаки доллара. Более того формулы можно писать в отдельной строке, для этого нужно проделать тоже самой, только обернуть все в двойной знак доллара.

## Изображения
Иногда в документы может понадобиться добавлять изображения для этого следует использовать блок figure, как показано ниже.
```LaTeX
\documentclass{article}

\usepackage{graphicx}

\begin{document}

\begin{figure}
  \includegraphics[width=\linewidth]{green.jpg}
  \caption{Green color}
  \label{fig:green}
\end{figure}


\end{document}
```
## Оглавление
Если в документе есть заголовки и подзаголовки, то можно добавить оглавление, котоое создается автоматически с помощью одной команды \tableofcontents.
```LaTeX
\documentclass{article}

\begin{document}

\tableofcontents
\newpage

\section{Section}
Lorem ipsum .....

\subsection{Subsection}
Lorem ipsum

\end{document}
```
## Библиография
С помощью расширения bibtex можно добавлять используемые источники, для этого нужно отедльно создать файл с расширеним .bib, который будет содержать источники в формате LaTeX. Далее в самом документе нужно добавить команды \bibliography, сообщающие LaTeX о расположении нашего файла .bib, и \bibliographystyle, которая выбирает один из различных библиографических стилей.

## Сноски
Создайть сноски можно с помощью команды \footnote и пометить их с помощью \label.

## Презентация
В LaTeX-е существует класс документа под названием Beamer. Это класс документов LaTeX, который используется для создания презентаций. Существует различные виды шаблонов, большиснтво из которых можно найти в галерее Overleaf, и набор интересных функций для создания красивых презентаций.

Здесь перечислены лишь некоторые из свойств и функций LaTeX-а, на самом деле существует множество других не менее инересных особенностей данной разметки. Более подробную информафию можно найти в официальное документации и на сайте overleaf.com.
